Courses

HTML5/CSS3

HTML or HyperText Markup Language is the standard markup language used to create web pages. CSS (Cascading Style Sheets) is a cornerstone technology used to create visually engaging user interfaces for web and mobile applications.

JavaScript/jQuery

JavaScript is the programming language of the web. It is bundled as part of web browsers and allow users interact with web and mobile apps. It is also used in server-side network programming and game development. jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML.

Git

Git is a distributed revision control system with an emphasis on speed, data integrity, and support for distributed, non-linear workflows.

Meteor

Meteor is a complete open source platform for building web and mobile apps in pure JavaScript, faster than you ever thought possible.

Frequently Asked Questions

What is The Code School?
The Code School is an 8-week intensive beginner-friendly coding school in Accra.

What will I learn at The Code School?
You will learn the fundamental technical skills needed to make beautiful web and mobile apps.

What's the schedule?
Classes hold from 8am to 5pm everyday.
Can you really become a programmer in 8 weeks?
Software engineering is a craft that takes years of deliberate practice and learning to master. We aim to graduate world-class beginners, and jumpstart your journey towards becoming an elite coder.

I don't know any programming. Is that ok?
Absolutely. No previous coding background is needed. We teach you everything from scratch. However, you do need to be proficient with a computer.
What happens after completing The Code School?
Incisia and The Code School will provide ongoing mentorship, support and employment recommendations for graduates.

Is this going to be easy?
No, this will be very hard work. But it should also be a lot of fun.

Do you offer any accreditation?
No we do not offer any accreditation, we are focused on graduating world class developer beginners whose accreditation is their own skill.
