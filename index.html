Frequently Asked Questions

What is The Code School?
The Code School is an intensive beginner-friendly coding school in Accra.

What will I learn at The Code School?
You will learn the fundamental technical skills needed to make beautiful web and mobile apps.

What's the schedule?
Classes hold from 9am till 12:30pm every weekday.

Can you really become a programmer in 10 weeks?
Software engineering is a craft that takes years of deliberate practice and learning to master. We aim to graduate world-class beginners, and jumpstart your journey towards becoming an elite coder.

I don't know any programming. Is that ok?
Absolutely. No previous coding background is needed. We teach you everything from scratch. However, you do need to be proficient with a computer.

Do I need to bring a laptop?
Yes. You need to bring your own computer.

What happens after completing The Code School?
Incisia and The Code School will provide ongoing mentorship, support and employment recommendations for graduates.

Is this going to be easy?
No, this will be very hard work. But it should also be a lot of fun.

How much does tuition cost?
Tuition at The Code School costs 500 GHS and 1,000 GHS for the Basic and Advanced courses respectively.

Where exactly will classes take place?
Classes will take place at iSpace, Osu.

Do you offer any accreditation?
No we do not offer any accreditation, we are focused on graduating world class developer beginners whose accreditation is their own skill.

What are my payment options?
Payment for each class should be made in full before class commences.
